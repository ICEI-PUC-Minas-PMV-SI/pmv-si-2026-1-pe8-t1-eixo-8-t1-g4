# Sistema de Solicitação de Materiais

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Trabalho de Conclusão de Curso`

`SEMESTRE: 8º`

O Sistema de Solicitação de Materiais tem como objetivo modernizar e otimizar o processo de solicitação de materiais entre os setores de Produção e Almoxarifado da empresa Novo Nordisk. Atualmente, o processo depende do uso de rádios comunicadores, o que pode gerar falhas de comunicação, retrabalho e lentidão nas operações.

Com a implantação do sistema, todas as solicitações passarão a ser registradas de forma centralizada, organizada e de fácil acesso para os usuários envolvidos. Isso proporcionará maior integração entre os setores, melhoria na comunicação, rastreabilidade das informações e redução de possíveis erros, tornando o processo mais ágil, seguro e eficiente.
